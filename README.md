# Excel & VBA Automation Portfolio

This repository presents a selection of Excel VBA automation projects
focused on improving efficiency, data quality, and business reporting.

The solutions showcased here were designed for consulting and real-world
business scenarios, including data cleaning, structured data entry, and
automated reporting.

## Core Skills Demonstrated
- Excel VBA (Macros)
- Process Automation
- Data Cleaning & Validation
- Excel UserForms
- Automated Reports & Dashboards
- Business-Oriented Data Analysis

### 2. Excel Data Entry Form

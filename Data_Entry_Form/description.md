## Excel Data Entry Form (UserForm)

### Problem
Manual data entry in spreadsheets often leads to inconsistent formats,
missing information, and frequent input errors.

### Solution
Designed a custom Excel UserForm using VBA that includes:
- Mandatory fields
- Dropdown lists
- Input validation rules
- Error messages for incorrect data
- Automated storage of validated records

### Tools & Techniques
- Excel VBA UserForms
- Data validation logic
- Event-driven programming

### Business Impact
- Standardized data entry process
- Reduced human error
- Improved data integrity for reporting and analysis

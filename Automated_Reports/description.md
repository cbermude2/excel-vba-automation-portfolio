## Automated Reporting Dashboard

### Problem
Manual generation of recurring reports is time-consuming and error-prone.

### Solution
Built an Excel-based automated reporting system that:
- Consolidates data from multiple sources
- Refreshes calculations automatically
- Generates dynamic tables and charts
- Produces management-ready reports with minimal manual effort

### Tools & Techniques
- Excel VBA
- Pivot Tables
- Automated refresh macros
- Dashboard design

### Business Impact
- Faster report generation
- Consistent reporting structure
- Improved decision-making through timely insights
